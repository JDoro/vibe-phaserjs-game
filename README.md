# Road Blocks Game

An exciting road blocks game built with PhaserJS and deployed to GitHub Pages.

## Game Features

- Endless runner gameplay
- Vehicle controlled by mouse or arrow keys
- Falling obstacles to avoid
- Progressive difficulty increase
- Score tracking based on obstacles avoided
- Game over when hitting an obstacle
- Restart functionality

## How to Play

- Move your mouse left/right to control the vehicle
- Or use arrow keys (left/right) to move
- Avoid the falling obstacles on the road
- Score increases for each obstacle you successfully avoid
- Game ends when you hit an obstacle!

## Development

### Prerequisites

- Node.js (v20 or higher)
- npm

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Deployment

This game is automatically deployed to GitHub Pages when changes are pushed to the main branch using GitHub Actions.

Play the game at: https://jdoro.github.io/vibe-phaserjs-game/

## Technologies

- [PhaserJS](https://phaser.io/) - Game framework
- [Vite](https://vitejs.dev/) - Build tool
- GitHub Actions - CI/CD
- GitHub Pages - Hosting
