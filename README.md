# Block Breaker Game

A classic block breaker game built with PhaserJS and deployed to GitHub Pages.

## Game Features

- Classic block breaker gameplay
- 5 rows of colorful blocks
- Paddle control with mouse movement
- Ball physics with collision detection
- Score tracking
- Lives system (3 lives)
- Win/lose conditions with restart functionality

## How to Play

- Move your mouse to control the paddle
- Keep the ball from falling off the bottom of the screen
- Break all the blocks to win
- You have 3 lives - don't let the ball fall!

## Development

### Prerequisites

- Node.js (v20 or higher)
- npm

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Deployment

This game is automatically deployed to GitHub Pages when changes are pushed to the main branch using GitHub Actions.

Play the game at: https://jdoro.github.io/vibe-phaserjs-game/

## Technologies

- [PhaserJS](https://phaser.io/) - Game framework
- [Vite](https://vitejs.dev/) - Build tool
- GitHub Actions - CI/CD
